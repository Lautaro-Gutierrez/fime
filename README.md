# FiMe - Tu Ecosistema de Finanzas Personales 🚀

FiMe es una aplicación web progresiva (PWA) diseñada para que cualquier persona pueda tomar el control total de su dinero. Desde la gestión de gastos diarios hasta el seguimiento de inversiones, FiMe centraliza tu vida financiera en un solo lugar, con una interfaz intuitiva y apta para todo público.

## ⚙️ Funcionalidades Principales
 Gestión de Finanzas Personales Control detallado de ingresos, gastos y presupuesto mensual para el usuario cotidiano.
 Módulo de Inversiones (Blotter) Herramienta integrada para usuarios que desean seguir sus Criptomonedas, Acciones (USAR), CEDEARs, Bonos y Plazos Fijos.
 P&L en Tiempo Real Visualización clara de la evolución del patrimonio.
 App Nativa (PWA) Instalable tanto en dispositivos móviles (Android) como en PC de escritorio.

## 🛠️ Stack Tecnológico y Desarrollo
Este proyecto es una demostración de desarrollo acelerado utilizando herramientas de Inteligencia Artificial. Fue conceptualizado, estructurado y desplegado en producción aplicando técnicas de Prompt Engineering avanzado

 Desarrollo Asistido por IA Claude Code y Antigravity.
 Frontend Next.js 16, React 19, Tailwind CSS v4.
 Backend y Base de Datos Supabase.
 Integración de APIs CoinGecko (Crypto), Finnhub (Mercado US), Data912 (Mercado AR), DolarAPI.
 Deployment Vercel.
